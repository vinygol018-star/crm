# **App Name**: FluxFlow CRM

## Core Features:

- Smart Outbound Dashboard: Real-time summary of leads, conversion rates, and revenue powered by Firestore real-time updates.
- Automated Pipeline Flow: System that automatically migrates leads between specialized views (Leads, Meetings, Sales, Lost) based on status updates.
- Intelligent Follow-up Tracker: Automated scheduling system that tracks contact levels and highlights leads needing urgent attention after 4 days of inactivity.
- AI Content Assistant tool: Uses AI to analyze lead history and niche to suggest personalized follow-up messages or conversation starters.
- Bulk Lead Ingestion: Fast data entry module allowing users to paste plain text lists or upload CSV files for mass lead creation.
- Financial Ledger System: Dedicated sales area to manage payments, installments, and operation faturamento tracking directly within Firestore.
- Audit Timeline: Visual history for each lead capturing status changes, meetings, and manual notes for full accountability.

## Style Guidelines:

- Primary Color: Vibrant Electric Blue (#1A73E8) to represent movement and professionalism.
- Background Color: Deep Charcoal (#0A0C0E), achieving the high-end SaaS dashboard aesthetic.
- Accent Color: Deep Violet-Blue (#5C6BC0) used for secondary interactive elements to create depth.
- Headline Font: 'Space Grotesk' for a technical, modern feel; Body Font: 'Inter' for high readability in dense lead lists and tables.
- Clean, thin-line icons with selective coloring: Green for faturamento, Red for urgency, and Blue for active contacts.
- Sidebar-driven navigation with expansive lead cards featuring smooth borders and subtle elevation shadows against the dark backdrop.
- Horizontal slide transitions when leads migrate between pipeline stages (e.g., from Sent to Meeting).
